# 🧩 Krystal's Personal Website

Check it out here! [https://your-website-link.vercel.app/](https://krystalpothilat.github.io/)

My personal interactive portfolio website designed as a **puzzle experience**. The pieces come together to form an image, while also serving as entry points into different sections showcasing my projects, experience, and interests.
This project blends **portfolio storytelling + interactive navigation**, turning browsing into an engaging visual experience.

## ✨ Features

### 🧩 Interactive Puzzle Navigation

- Website is structured as movable puzzle pieces
- Drag and reposition pieces to complete the full image
- Pieces represent a section of the site (Projects, Experience, About, etc.)
- Completing the puzzle reveals the full portfolio layout

### 🧭 Click-to-Navigate Sections

- Don’t want to play? No problem
- Click pieces to navigate and read more about me without completing the puzzle
- You can still explore the same information without assembling everything first

## 🧠 Why I Built This

I wanted to move away from the typical “scroll and read” portfolio format and build something more interactive.
The goal wasn’t just to present information, but to make exploring it feel more engaging and less static.
